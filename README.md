# plusコマンド
* 内容： 標準入力から読み込んだ数字を足す。
* インストール方法: Pythonの利用できる端末上でファイル(plus)をコピーする

* 実行例： $seq 10 | ./plus
* 実行後： 55　(1から10を順に足し合わせた)

## 必要なソフトウェア
* Python：3.7∼3.10

## テスト環境
